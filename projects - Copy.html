PROJECTS

PRINGLE SEARCH

A search engine built from the ground up in Python.

I built this search engine last winter for an Information Retrieval course. This involved creating an index that related terms to documents they appeared in. In order to meet the strict time constraints (return results under 300ms) I had to split the documents in my collection alphabetically and create a second index that would track where terms appeared in the first index. This allowed me to only load the line containing information for the term I was interested in.

Python was used as the backend scripting language that would host the webserver (specifically Flask). Jinja was used as my templating language.

SMART MIRROR

A smart mirror powered by Python, running on a Raspberry Pi.

I built this "smart" mirror the summer of my freshman year of college. The main objective was to build something I'd want to use using the skills I had learned that year in college. To complete this project I first purchased a Raspberry Pi that would serve as the core of the project. I then stripped an old monitor down to its LCD screen and built a housing for the components out of a simple wooden frame. After writing a script to handle the logic of the mirror I purchased a two-way mirror and put everything together.

The primary logic was written in Python and the GUI was built using Tkinter.

PEER VIEW

A video calling service where users can connect using only a peer id.

My project this summer was to build a something I could see myself using. The result? A website where users can connect using a 6 digit id and communicate via video chat, a messaging system, and even screenshare. This project was built using Peer JS an API that makes use of WebRTC to send data through your browser.

To build this site I used HTML, CSS, Bootstrap4 and Node.JS as the backend scripting language. Peer JS was the API this website was built on top of.

WORD ROOTS

An Android application for non-native English speakers and the curious alike.

This project was built the summer going into my freshman year of college. Having only learned Java from my APCS course at this point, I wanted to build a project that was familiar but vastly different from anything I had done in class. After a hours of tinkering and research the result was a somewhat shoddy, but completely functional word game application. The core premise of the game was to come up with as many words from a root word as possible. The game would award a corresponding amount of stars depending on how well the player did.

The application was written using Android Studio with the development language used being Java.
